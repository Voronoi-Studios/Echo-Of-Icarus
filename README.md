<p style="text-align:center">**Echo of Icarus** is a compact standalone experience inspired by the larger vision of the Icarus server. Explore layered procedurally generated dungeon floors, fight monsters, uncover hidden routes, and gather loot that helps you push deeper into the maze. Each run reveals new paths, new dangers, and new echoes waiting below.</p>

![](https://media.forgecdn.net/attachments/description/1526293/description_0bec54bb-4cd2-433a-ae02-3e5dd8ca4992.png)
